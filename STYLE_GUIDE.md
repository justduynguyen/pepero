## Color Palette

### Primary Colors
- **Pink 500** (#FF6B9D) - Main brand color
- **Purple 400** (#AB47BC) - Secondary brand color
- **Text Primary** (#2D1B2E) - Main text & borders

### Background
- **BG Primary** (#FFF5F7) - Light pink
- **BG Secondary** (#FFE5EC) - Soft pink
- **Purple 50** (#F3E5F5) - Light purple for gradients

### Text Colors
- **Primary** (#2D1B2E) - Headings & important text
- **Secondary** (#5A3D5C) - Body text
- **Muted** (#8B7E8C) - Secondary info

## Typography

### Fonts
- **Headings**: Fredoka (Google Fonts) - Fun, rounded, retro
- **Body**: Inter (Google Fonts) - Clean, readable

### Sizes
- **H1**: 24-32px, bold
- **H2**: 20-24px, bold  
- **H3**: 18-20px, semibold
- **Body**: 14-16px, regular
- **Small**: 12-14px, regular

## Components

### Buttons (Retro Style)
- Border: 2-3px solid #2D1B2E
- Box Shadow: 3-4px 3-4px 0 rgba(45,27,46,0.3)
- On hover: translate(2px, 2px) + reduce shadow
- On active: translate(4px, 4px) + no shadow
- Background: Gradient or solid color

### Cards (Retro Style)
- Border: 3-4px solid #2D1B2E
- Box Shadow: 6-8px 6-8px 0 rgba(45,27,46,0.15)
- Border Radius: 16-24px
- Background: White or light pink

### Inputs
- Border: 2px solid pink-200 normal, pink-500 focus
- Border Radius: 12px
- Padding: 12-16px
- Focus ring: 2px pink-500

## Gradients
